PIN MAPPING TABLE
================================================================================

ESP32-S3 Min Board
├─ GPIO 8   → I2C SDA (Serial Data)
├─ GPIO 9   → I2C SCL (Serial Clock)
├─ 3V3      → Power Supply (3.3V)
└─ GND      → Ground Reference

QMC5883L (GY-271 Module)
├─ VCC      → ESP32 3V3 (3.3V)
├─ GND      → ESP32 GND
├─ SDA      → ESP32 GPIO 8
├─ SCL      → ESP32 GPIO 9
└─ Address  → 0x0D (I2C)

SSD1306 OLED (64x32 Display)
├─ VCC      → ESP32 3V3 (3.3V)
├─ GND      → ESP32 GND
├─ SDA      → ESP32 GPIO 8
├─ SCL      → ESP32 GPIO 9
└─ Address  → 0x3C (I2C)
